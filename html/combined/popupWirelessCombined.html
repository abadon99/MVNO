1. 결합 혜택
  • 결합된 요금제에 따라 무료 결합 데이터가 제공됩니다.(일부 요금제는 결합은 가능하나 무료 데이터가 미제공됩니다. 결합 혜택 대상 요금제는 이벤트 페이지 및 자세히보기를 참고해주세요)
  • 중복 결합이 된 회선에는 무료 결합 데이터가 중복 제공되지 않습니다. (ex.A회선이 B회선, C회선과 각각 결합이 되어 있을 경우 A회선은 데이터 중복 수령 불가)
  • 결합 시 첫 데이터는 별도의 신청 없이 즉시 제공되며, 이후에는 매월 1일에 데이터가 제공됩니다.
  • 결합 해지 시 매월 제공되던 데이터는 중단됩니다.
  • 결합 시 제공되는 데이터 명칭은 『LTE 데이터 추가 제공(결합)』이며, 데이터 FREE 쿠폰과 중복 적용하여 사용 가능합니다.-데이터 소진순서 : LTE 데이터충전 > 데이터 Free 쿠폰(20GB의 경우) > 데이터 Free 쿠폰 10GB > LTE 데이터 추가제공 (24개월) > LTE 데이터 추가제공 20GB(결합) > 요금제기본데이터 > 데이터 Free 쿠폰 50GB(1회) > QoS
  • 결합 시 제공되는 데이터는 테더링으로 사용 가능하며, 함께쓰기, 데이터쉐어링, 로밍으로는 사용 불가합니다.
